Product Specification Document
GKI Kutisari Management System
1. Product Overview
1.1 Product Name
Sistem Informasi GKI Kutisari (GKI Kutisari Information System)

1.2 Product Description
A comprehensive web-based church management system designed for GKI Kutisari Indah (Gereja Kristen Indonesia - Indonesian Christian Church) to manage church resources, assets, rooms, transportation, and member activities efficiently.

1.3 Target Users
User Role	Description
Management	Church administrators with full access to manage resources, users, and website content
Members	Church congregation members with access to borrow items, reserve rooms, and request transportation
1.4 Platform
Type: Progressive Web Application (PWA)
Deployment: Vercel (Serverless Functions)
Database: Supabase (PostgreSQL)
Frontend: HTML, TailwindCSS, Vanilla JavaScript
